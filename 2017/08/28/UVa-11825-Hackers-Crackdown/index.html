描述

一个黑客入侵一个有 $n$ 台计算机 $(1, 2, \cdots n - 1)$ 的网络，一共有 $n(1 \leq n \leq 16)$ 种任务，每台计算机都运行这所有的任务。对于每台计算机可以选择一项服务，终止这台计算机与其相邻计算机之间的这种服务，你需要让尽量多的服务瘫痪。
